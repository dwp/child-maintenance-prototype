Calculate child maintenance

Use this calculator to work out child maintenance for your circumstances.

The receiving parent has main day-to-day care of the child. The paying parent does not have main day-to-day care.

The calculator uses the same rules as the Child Maintenance Service.

You'll need to know:

  • whether the paying parent gets any government benefits or State Pension
  • the paying parent's total income, before tax is taken off
  • if any other children live with the paying parent

If the paying parent has child arrangements with more than one receiving parent a calculation should be done for the total number of children, and divided equally between them.
